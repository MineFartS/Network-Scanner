# Network Scanner

A simple and lightweight network scanner.

Checks the status of all IPs on the lan

## Usage:
```
irm https://raw.githubusercontent.com/MineFartS/Network-Scanner/refs/heads/main/run.ps1 | iex
```
## Example:

```
InterfaceIndex InterfaceAlias
-------------- --------------
             8 Wi-Fi


InterfaceIndex:

```
